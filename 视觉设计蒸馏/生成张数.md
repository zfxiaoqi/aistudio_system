## 评分标准

### 居家松弛场景评分标准

总分100分，低于85分建议重新生成。

| 评分项 | 权重 | 合格标准 |
| --- | --- | --- |
| 人物真实度 | 20 | 五官自然、皮肤真实、手脚正确、成人身份明确 |
| 松弛情绪 | 15 | 动作自然、肩颈放松、表情不营业 |
| 空间视觉基因 | 15 | 暖白、浅木、大窗、薄纱帘、圆角家具统一 |
| 光影质量 | 15 | 窗光方向明确、暖而不黄、高光阴影有细节 |
| 产品准确度 | 20 | 版型、材质、颜色、包装与参考一致 |
| 构图与空间 | 10 | 人物完整、层次清楚、留白得当、无畸变 |
| 清晰度与材质 | 5 | 发丝、皮肤、织物、木纹清楚且不过锐 |

#### 一票否决项
- 人物看起来未成年或年龄不明确
- 出现多手、多脚、粘连手指、肢体融合或复制脸
- 产品版型、颜色、包装结构明显错误
- 画面出现色情化、挑逗化或情趣内衣表达
- 人物与家具、地板、道具无真实接触，重力关系错误
- 整体变成浓黄滤镜、冷灰棚拍或杂乱低价电商风

### 海边出行场景评分标准

总分100分，建议85分以上才进入商业素材库；任一硬性否决项出现则直接重生成。

| 评分项 | 分值 | 通过标准 |
| --- | --- | --- |
| 品牌情绪 | 20 | 同时具备轻松、洁净、呼吸感、自然生命力 |
| 场景真实度 | 15 | 天空、海水、礁石/沙滩自然，非贴图或蓝幕感 |
| 人物自然度 | 15 | 22–30岁亚洲女性，五官、皮肤、动作、表情真实 |
| 光线一致性 | 10 | 人物与环境受光统一，白衣不过曝，肤色不灰 |
| 构图与留白 | 10 | 低海平线、留白合理、人物与文案区不冲突 |
| 空间层次 | 10 | 前中远景清晰，透视、遮挡与空气感正确 |
| 产品还原度 | 10 | 版型、颜色、纹理、包边与比例准确 |
| 技术清晰度 | 5 | 面部、手指、发丝、衣物和产品清晰无错误 |
| 双人关系准确度 | 5 | 仅双人图评分：差异明确、互动自然、无复制与粘连 |

#### 硬性否决项
- 人物不是22–30岁亚洲年轻女性
- 手、脸、腿或身体结构明显错误
- 双人出现复制脸、融合肢体或关系僵硬
- 产品版型、颜色或关键结构明显改变
- 皮肤强塑料感，背景明显为蓝幕/贴图/抠图
- 画面产生色情、挑逗或不适当身体凝视
- 海边旅途感不成立，退化为普通泳装、影棚或旅游合影

## 自动纠偏规则

| 常见偏差 | 自动补救指令 |
| --- | --- |
| 画面太像旅游打卡 | 增加大留白、降低道具、人物不看镜头、改为行走或望海抓拍 |
| 人物太僵硬 | 放松肩颈、身体转15°–45°、重心落在一条腿、加入轻微风感动作 |
| 皮肤太假 | 降低磨皮与高光，恢复毛孔、绒毛和自然色差，统一面部与身体肤色 |
| 天空太艳 | 降低蓝色饱和度，增加浅蓝渐变和空气透视，避免纯色背景感 |
| 海景像贴图 | 降低背景虚化，补充浪花、远近水色、海面反光与前景遮挡关系 |
| 双人像复制人 | 改变发型、脸型细节、服装色、前后位置、动作与视线方向 |
| 双人互动不自然 | 减少接触面积，使用牵手、轻搭肩、并肩走或回头回应 |
| 画面太健身 | 删除拉伸和肌肉强调，改为步行、迎风、望海或自然展开手臂 |
| 产品不清晰 | 调整人物角度与外搭位置，避免遮挡，恢复产品轮廓、面料和包边细节 |
| 白色服装过曝 | 降低高光，增加细腻灰阶与自然褶皱，保留纤维和边缘层次 |
| 画面不够有生命力 | 增加细碎浪花、海面反光、发丝与衣摆动态、自然露齿笑 |
