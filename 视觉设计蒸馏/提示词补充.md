## 生成前检查清单

- [ ] 人物是否明确为22–30岁成年亚洲女性？
- [ ] 单人或双人逻辑是否清楚？
- [ ] 双人是否有发型、脸型、动作和空间层次差异？
- [ ] 动作是否来自真实居家行为并符合受力？
- [ ] 空间是否至少包含暖白、浅木、大窗自然光中的两项？
- [ ] 家具与道具是否克制且服务情节？
- [ ] 光线是否有方向，是否暖而不黄？
- [ ] 人物与空间光向是否一致？
- [ ] 是否保留皮肤、面料、木纹与窗帘材质？
- [ ] 产品版型、包装结构和品牌色是否准确？
- [ ] 构图是否保留呼吸留白，手脚是否完整？
- [ ] 是否加入完整负面提示词？
