# 色调

## 米色调

### 核心特征
- 奶油白、米白、米杏色、浅裸色
- 柔和肤色、低饱和、高明度
- 轻盈空气感、温暖安心感

### 米色调色板
| 颜色角色 | 建议色值 | 用途 |
| --- | --- | --- |
| 核心白 | #F7F3EA | 墙面、窗帘、床品、服装 |
| 奶油米 | #E9DDC9 | 家具、软装、空间过渡 |
| 浅木色 | #C9A778 | 地板、边几、少量家具 |
| 暖肤色 | #D9A47F | 人物皮肤参考 |
| 杏桃点缀 | #E9A66F | 挂画、苹果、微小装饰 |
| 自然绿 | #6F8060 | 绿植或窗外树叶 |

### 配色比例
- 暖白/奶油白：55%–65%
- 浅木/燕麦色：20%–30%
- 人物肤色与服装：10%–20%
- 杏桃/绿色/品牌蓝点缀：不超过10%

### 影调关键词
- 奶油米白、柔光棚拍、低对比度、高明度
- 温暖、安心、放松、亲肤、有治愈感
- 精致但不端着、干净舒适的生活状态

### 适用场景
- 居家松弛场景
- 米色自在系室内影调
- 柔软亲肤卖点表达
- A类室内母版

## 蓝色调

### 核心特征
- 天空蓝、海水蓝、浅天蓝
- 白色、浅雾灰、低饱和肤色
- 高明度、低饱和、空气感强

### 蓝色调色板
| 颜色角色 | 建议色值 | 用途 |
| --- | --- | --- |
| 品牌清爽蓝 | #2899FF | 少量品牌联想、配件点缀 |
| 天空浅蓝 | #B9DCF3 | 大面积背景基底 |
| 海水蓝 | #4F9FC8 | 中景海面 |
| 远海深蓝 | #247DA8 | 远景层次 |
| 洁净白 | #F8F8F4 | 服装、毛巾、浪花 |
| 雾霾浅蓝 | #C9DCE9 | 外搭与柔和层次 |
| 浅柔粉 | #F2CCD3 | 女性化轻量点缀 |
| 岩石灰 | #646565 | 前景稳定与自然对比 |

### 配色比例
- 蓝色环境：45%–60%
- 天空浅色：20%–35%
- 白色：15%–25%
- 肤色：8%–15%
- 点缀色：≤5%

### 影调关键词
- 蓝白主调、清透自然光、开阔天空、大面积留白
- 清爽、安全、轻盈、透气、有秩序、值得信赖
- 舒适型洁净蓝、干净专业但不冰冷

### 适用场景
- 海边出行场景
- 蓝色洁净棚拍
- 产品功能证据化展示
- A类海边影调

## 色调使用原则

### 米色调使用原则
- 画面整体偏暖中性，白色不能漂蓝，也不能发黄发脏
- 绿色仅用于增加生命力，不形成植物园式背景
- 品牌蓝仅在产品包装或少量品牌识别中出现，避免破坏奶油暖调
- 饱和度保持低至中等

### 蓝色调使用原则
- 蓝色不是冷科技蓝，而是舒适型洁净蓝
- 避免高饱和艳色、脏灰色、网红滤镜色、厚重暗调
- 人物肤色自然偏暖，环境清爽偏冷，形成冷暖平衡
- 不做成医疗器械感或冰冷实验室感

### 跨场景色调衔接
- 米色调：居家松弛、米色自在、居家柔弹
- 蓝色调：海边自在、蓝色洁净、蓝色棚拍功能证据
- 两种色调可通过浅蓝与米白的过渡色进行衔接
