# 巴迪高海边出行场景视觉基因

> 适用范围：巴迪高 AI Studio 海边出行视觉生成应用  
> 视觉对象：22–30 岁亚洲年轻女性，支持单人 / 双人画面  
> 核心任务：以海边旅途为情绪场景，稳定生成兼具轻松、洁净、呼吸感、自然生命力与商业质感的品牌画面  
> 版本：V1.0

---

## 1. 品牌视觉总纲

### 1.1 一句话视觉定义

**在开阔、真实、洁净的海边自然环境中，以蓝天、海风、日光和舒展的人物状态，表现年轻女性出行时卸下负担、自在呼吸的生命感。**

### 1.2 核心情绪公式

```text
海边旅途感 = 开阔天空 × 真实海风 × 清透日光 × 松弛人物 × 蓝白洁净 × 自然互动
```

### 1.3 品牌核心感受

| 核心感受 | 画面表达 | 禁止偏移 |
| --- | --- | --- |
| 轻松 | 肩颈舒展、自然走动、迎风、轻笑、随性互动 | 僵硬站军姿、强凹造型、过度时尚冷脸 |
| 洁净 | 蓝白主色、清澈天空、干净皮肤、克制道具 | 浑浊灰天、脏乱沙滩、复杂游客背景 |
| 呼吸感 | 50% 左右大留白、低海平线、衣摆与发丝被风带动 | 人物塞满画面、背景拥堵、重压迫构图 |
| 自然生命力 | 正午偏柔阳光、真实动作瞬间、海浪反光、轻盈笑容 | 健身硬核感、夸张蹦跳、虚假网红笑 |
| 出行感 | 行李箱、轻薄外搭、步行、抵达、同行关系 | 重度度假装饰、摆拍式旅行打卡 |

### 1.4 视觉气质权重

```text
自然原生 30% + 清爽洁净 25% + 松弛自在 20% + 轻盈生命力 15% + 商业精致度 10%
```

## 2. 参考大片的视觉基因提取

### 2.1 必须保留的稳定基因

1. **天空是第一视觉空间**：浅蓝到天蓝的自然渐变天空占画面约 45%–70%，云量少或仅有极淡薄云。
2. **海平线偏低**：海平线通常位于画面下方 1/4–2/5，释放上方呼吸空间。
3. **蓝、白、肤色为主体色**：天空蓝、海水蓝、纯净白、浅雾霾蓝与自然暖肤色共同构成画面。
4. **光线真实明亮**：接近晴朗上午 9:00–11:00 的自然日光，通透但不过曝，人物面部保有柔和补光。
5. **风是隐形叙事线**：发丝、衬衫、毛巾、轻薄布料形成同方向流动，画面因此具有“正在旅途中”的即时感。
6. **人物状态先于姿势设计**：动作像被摄影师抓拍到的真实瞬间，而非为了展示身体刻意摆拍。
7. **空间层次清晰**：岩石 / 湿沙前景、人物与海浪中景、海面与天空远景，形成稳定的三层纵深。
8. **画面干净但不失自然纹理**：保留岩石、海浪、织物、皮肤的真实细节，不磨成塑料质感。
9. **商业画面有留文案空间**：人物常偏左或偏右，另一侧保留连续、干净的天空或海面。

### 2.2 参考画面的构图DNA

| 构图类型 | 人物位置 | 景别 | 视觉作用 | 推荐场景 |
| --- | --- | --- | --- | --- |
| 大留白全身 | 左 / 右侧三分之一 | 全身、远中景 | 最强呼吸感与品牌感 | KV、首页横幅 |
| 低机位舒展 | 中央偏侧 | 3/4 身、全身 | 强调天空、自由和生命力 | 单人主视觉 |
| 海浪环境肖像 | 中央或偏侧 | 3/4 身 | 人物与自然平衡 | 产品穿着展示 |
| 双人近景互动 | 前后错位 | 半身、3/4 身 | 表现同行关系与快乐 | 社交内容、详情页 |
| 旅途道具入镜 | 人物偏侧，道具在下方 | 全身 | 强化“抵达海边”的出行叙事 | 行李箱 / 旅行用品场景 |

### 2.3 镜头语言

- 推荐焦段：35mm、50mm；双人近景可使用 50mm–70mm。
- 推荐机位：平视或轻微低机位，镜头高度约腰部至胸口；不使用极端仰拍。
- 景深：环境清晰度适中，人物清晰、远海略柔；避免强烈人像虚化导致“假海景”。
- 快门感：凝固人物清晰轮廓，同时保留衣摆和发丝的轻微动态。
- 透视：自然真实，不拉长四肢，不夸大头身比，不用超广角边缘畸变。

## 3. 海边场景视觉规范

### 3.1 场景母版

```text
真实自然海岸，开阔浅蓝天空，清澈蓝绿色海面，细碎白色浪花，低海平线，
前景为干净自然的礁石、湿沙或浅水，远处可有低饱和海岛轮廓，
空气通透，天气晴朗，游客稀少或完全无人，整体洁净、安静、可呼吸。
```

### 3.2 可选子场景

| 场景编号 | 场景 | 关键元素 | 情绪定位 | 推荐人数 |
| --- | --- | --- | --- | --- |
| S01 | 海边礁石 | 深灰自然礁石、浪花、水面反光 | 原生、清醒、自由 | 单人 / 双人 |
| S02 | 浅水沙滩 | 细沙、浅水波纹、轻柔海浪 | 轻盈、柔和、亲近 | 单人 / 双人 |
| S03 | 海岸高地 | 岩壁、低矮海岸植物、远海 | 抵达、启程、旅途感 | 单人优先 |
| S04 | 海边步道 | 简洁浅色步道、海风、远海 | 日常出行、轻快 | 单人 / 双人 |
| S05 | 海边民宿外 | 白墙、薄帘、露台、海景 | 洁净、舒适、休憩 | 双人优先 |
| S06 | 清晨海滩 | 淡蓝天空、柔和侧光、平静海面 | 安静、治愈、松弛 | 单人优先 |

### 3.3 天空规范

- 色彩：浅天蓝到清澈蔚蓝的真实渐变，上深下浅。
- 云量：0%–15%，可无云或仅有稀薄白云。
- 亮度：明亮通透，高光柔和，不出现纯白死区。
- 空气：轻微海风带来的清透感，远景存在自然空气透视。
- 禁止：戏剧性乌云、火烧云、紫色晚霞、热带明信片式浓艳天空、棚拍蓝色背景布。

### 3.4 海面规范

- 色彩：自然天蓝、蓝绿或稍深的远海蓝；由近到远略有层次变化。
- 状态：小到中等海浪，浪花洁白，海面有细碎自然反光。
- 水体必须有真实纹理、方向和深度，不应像塑料、玻璃或平面贴图。
- 禁止：巨浪、风暴海、过饱和宝石蓝、荧光青、完全镜面无纹理水面。

### 3.5 前景与环境细节

- 礁石：深灰、灰褐或冷灰，自然湿润纹理，形态克制，不尖锐压迫。
- 沙滩：细腻、干净、浅米灰色，不出现垃圾、脚印密集区或杂乱设施。
- 植物：仅少量低矮海岸植物，低饱和灰绿色，作为真实环境点缀。
- 道具：银灰行李箱、白色毛巾、轻薄白衬衫、小型旅行袋；每幅画面建议 0–2 件。
- 禁止：遮阳伞、躺椅、泳圈、椰树、酒杯、彩旗同时堆叠形成度假广告感。

## 4. 人物视觉规范

### 4.1 统一人物设定

```text
22–30 岁亚洲年轻女性，具有当代东亚都市女性气质；
五官自然耐看而非网红模板脸，面部轮廓柔和，比例真实；
身形健康匀称、自然有活力，不追求极瘦、夸张曲线或健身肌肉感；
呈现真实旅行状态中的轻松、自信、洁净与生命力。
```

### 4.2 皮肤质感

- 自然健康的浅暖肤色或中性偏暖肤色，面部与身体色温统一。
- 可见细腻毛孔、轻微绒毛、自然肤理与真实高光。
- 面颊、肩颈、手臂可有非常轻微的阳光暖意，但不晒伤、不油亮。
- 阴影中仍保留肤色层次，避免灰脸、脏脸或局部过度锐化。
- 不使用强磨皮、美颜滤镜、蜡像皮、陶瓷皮、假白肤色。

### 4.3 五官与表情

- 五官：自然东方骨相，眉眼清爽，鼻唇比例真实，不做夸张欧式深邃脸。
- 表情主选：轻笑、自然露齿笑、闭眼感受海风、侧望海面、与同伴相视而笑。
- 情绪：愉悦但克制，像刚抵达海边后的放松，而非商业主持式大笑。
- 眼神：有明确落点，可看向海、远方、同伴或轻微看镜头；避免空洞失焦。

### 4.4 妆造

- 妆容：轻薄底妆、原生眉、低饱和蜜桃或裸粉唇、淡淡腮红、自然睫毛。
- 质感：无明显修容、无闪片、无浓重眼线、无红毯妆或韩式水光过亮效果。
- 饰品：极简小耳钉、细耳圈或完全无饰品；每人最多 1–2 种。
- 美甲：短甲、透明或裸粉色；避免长甲、钻饰和彩色复杂图案。

### 4.5 发型与风感

- 推荐发型：自然黑色或深棕黑色短直发、锁骨发、自然长直发、低马尾。
- 双人建议形成差异：一人利落短发 / 锁骨发，一人柔和长发 / 低马尾。
- 发丝应被海风轻轻吹动，方向与衣摆一致；保留少量碎发增强真实感。
- 禁止：高难度盘发、精致大卷、湿发造型、夸张漂染、每根头发过度整齐。

### 4.6 服装与产品呈现

- 主色：纯净白、浅天蓝、雾霾蓝、浅粉、米白。
- 推荐：白色背心、轻薄白衬衫、浅蓝外搭、白色休闲短裤、巴迪高一次性内裤或轻柔贴身衣物。
- 面料：细腻棉感、轻薄梭织、微透但不暴露的罩衫、自然垂坠与真实褶皱。
- 造型层次：贴身基础层 + 轻薄外搭，利用海风产生动态和呼吸感。
- 产品必须保留真实版型、腰高、裤脚弧线、包边、纹理、颜色与面料厚度，不随人物姿势任意变形。
- 穿着表达健康、自然、洁净，不情色化，不使用挑逗姿势或刻意身体特写。

## 5. 单人画面规则

### 5.1 单人人物状态

```text
像独自抵达海边、正在感受风与光的年轻旅行者；
内在放松、自信、轻盈，不对镜头表演，不显孤独。
```

### 5.2 推荐动作

1. 迎风慢走，一只脚自然迈出，手臂轻微摆动。
2. 站在礁石或浅水边，身体略转 15°–45°，侧头望向远海。
3. 双臂自然打开或轻微伸展，肩颈放松，不做健身拉伸。
4. 手持轻薄白布 / 毛巾，让布料顺风展开。
5. 一手轻提行李箱拉杆，呈现“刚刚抵达”的旅行瞬间。
6. 轻抚被风吹动的发丝，微笑或闭眼感受日光。
7. 外搭从一侧肩膀自然滑落，动作松弛但不过度暴露。

### 5.3 单人构图

- 横图：人物放在左 / 右三分之一，面朝留白区域；天空留白约 50%–65%。
- 竖图：人物可略偏中，头顶保留 15%–25% 天空，下方保留完整腿部或自然裁切至大腿。
- 人物占画面：全身 25%–45%；3/4 身 45%–65%；不建议人物占满 80% 以上。
- 动作线条：形成轻微斜线、S 形或向外展开的开放线条。

### 5.4 单人禁止项

- 双手叉腰、挺胸收腹式标准模特站姿。
- 直挺挺看镜头、表情空白、身体左右完全对称。
- 跑跳幅度过大、瑜伽动作、健身拉伸、舞蹈姿势。
- 身体比例被拉长，腿部超过真实比例，脚踩礁石缺乏重力关系。

## 6. 双人画面规则

### 6.1 双人关系定义

```text
两位 22–30 岁亚洲年轻女性朋友共同出行；
关系自然、平等、亲近，像旅途中真实交流与玩笑的瞬间；
画面重点是“有人同行的轻松”，不是闺蜜模板照或刻意亲密表演。
```

### 6.2 双人差异化设置

| 维度 | 人物 A | 人物 B |
| --- | --- | --- |
| 发型 | 短发 / 锁骨发 | 中长发 / 低马尾 |
| 服装色 | 浅蓝 + 白 | 米白 / 浅粉 + 白 |
| 位置 | 中景或画面主人物 | 前景轻虚化或侧后方 |
| 动作 | 主动发起互动 | 自然回应互动 |
| 神态 | 开朗轻笑 | 温柔微笑 / 侧望同伴 |

### 6.3 推荐互动动作

1. 两人牵手抬起形成自然弧线，一人看向同伴，一人轻笑。
2. 并肩走向海边，步伐错开，衣摆与发丝同向飘动。
3. 前后轻微错位，一人回头看同伴，另一人自然跟上。
4. 两人一起展开白色毛巾或轻薄布料，让海风形成视觉连接。
5. 一人扶行李箱，另一人轻靠或伸手接过旅行物品。
6. 轻搭肩、轻挽手臂、自然贴近交谈；接触应松弛而非用力拥抱。
7. 一人位于前景局部入画，另一人清晰处于中景，形成真实抓拍感。

### 6.4 双人构图

- 优先使用前后错位、高低错位、动静错位，避免两人平行站成一排。
- 两人之间保留适度负空间，肢体接触点必须清晰、自然、无手指粘连。
- 横图中双人整体占画面 35%–60%，仍需保留大面积天空或海面。
- 竖图可使用一近一远：前景人物占 30%–45%，主人物占 40%–60%。
- 面部不能互相遮挡，肢体不能穿插，服装轮廓必须可辨识。

### 6.5 双人禁止项

- 两人长相、发型、身形、服装完全复制。
- 两人同时直视镜头并做同一动作，形成旅游合影感。
- 过度拥抱、贴脸、背人、夸张旋转，抢走产品与出行主题。
- 手臂数量错误、手指粘连、手掌穿过衣物、前后景比例失真。

## 7. 光线视觉规范

### 7.1 主光设定

```text
晴朗海边上午的自然阳光，光线从画面左前方或右前方 30°–60° 照射；
人物面部有柔和自然补光，肤色清透，鼻侧与下颌存在轻微真实阴影；
头发、肩颈与白色衣物边缘带有干净的日光轮廓；
海面出现细碎高光，但不大面积过曝。
```

### 7.2 光感比例

| 项目 | 建议 |
| --- | --- |
| 整体曝光 | 明亮 +0.3 至 +0.7 EV 的视觉感受 |
| 对比度 | 中低对比，保留岩石与衣物细节 |
| 高光 | 清透但不溢出，白色服装仍有纹理 |
| 阴影 | 冷而不灰，面部阴影轻柔 |
| 色温 | 中性略暖的人物 + 清爽偏冷的环境 |
| 饱和度 | 中低饱和，蓝色清澈但不浓艳 |

### 7.3 禁止光效

- 影棚硬闪光、强烈环形灯眼神光。
- 金色黄昏、橙红逆光、赛博蓝光、电影式浓重青橙调色。
- 人物与背景光向不一致，身体边缘出现抠图白边。
- 白衣大面积死白、皮肤油亮反光、面部阴影发黑。

## 9. 构图、空间与版式习惯

### 9.1 三层空间

```text
前景：礁石 / 湿沙 / 人物局部 / 旅行道具
中景：主体人物、同行互动、海浪
远景：海平线、远岛、天空
```

- 三层之间需要清晰遮挡与大小关系。
- 海平线保持水平，除非极轻微动态倾斜，不使用明显荷兰角。
- 前景纹理用于增强真实感，但不遮挡产品主体。
- 远景需有空气透视，不能与人物同等锐利。

### 9.2 留白规则

- 横版 KV：留白区 45%–65%，优先为纯净天空。
- 竖版详情页：留白区 25%–45%，可位于人物头顶或侧方。
- 文案安全区：连续干净、无人物肢体、无复杂浪花或山体穿过。
- 人物视线与动作方向应朝向留白区，形成视觉延伸。

### 9.3 画幅建议

| 画幅 | 用途 | 构图重点 |
| --- | --- | --- |
| 16:9 | 官网横幅、品牌 KV | 大天空、低海平线、人物偏侧 |
| 3:4 | 天猫详情页、社媒竖图 | 3/4 身或全身、保留环境 |
| 4:5 | 社交平台信息流 | 人物与产品平衡，顶部留白 |
| 1:1 | 商品卡片、内容封面 | 双人互动或半身环境肖像 |

### 9.4 裁切规则

- 优先完整保留头部、手、产品和关键互动点。
- 若裁切肢体，应在大腿中段、腰上或上臂等自然位置，避免切手腕、脚踝、膝盖、颈部。
- 产品穿着画面不得因构图裁掉关键腰头、裤脚或面料区域。

## 10. 清晰度与真实质感标准

### 10.1 清晰度目标

- 商业级高清摄影，建议最终输出长边不低于 3000px。
- 主体面部、眼睛、发丝、手指、衣物边缘、产品纹理清晰。
- 天空与海面保留自然渐变，不产生色带、噪点或压缩块。
- 礁石与海浪有真实细节，但锐度低于主体人物。
- 不使用全画面过度锐化，不产生皮肤毛孔夸张、边缘白线或 HDR 质感。

### 10.2 真实度检查

- 人物脚部与地面存在合理接触、重心和投影。
- 海风方向在头发、衣摆、毛巾和浪花上基本一致。
- 人物受光方向与岩石、海面受光一致。
- 肌肤、织物、礁石、海水具有不同材质反射特征。
- 双人互动时手指数量、关节方向、前后遮挡正确。
